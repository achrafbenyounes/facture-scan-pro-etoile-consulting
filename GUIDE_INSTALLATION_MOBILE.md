# 📱 Guide d'installation — Application mobile (PWA)

FactureScan Pro est une **Progressive Web App (PWA)** : elle s'installe
directement depuis le navigateur, sans passer par l'App Store ou Google Play.

---

## 🤖 Android (Chrome) — Recommandé

1. Ouvrez l'URL de votre application dans **Google Chrome**
2. Un **bandeau** apparaît automatiquement en bas : *"Installer FactureScan Pro"*
3. Appuyez sur **Installer**
4. L'icône apparaît sur votre écran d'accueil ✅

> Si le bandeau ne s'affiche pas : menu Chrome (⋮) → **"Ajouter à l'écran d'accueil"**

---

## 🍎 iPhone / iPad (Safari) — Obligatoire

> ⚠️ Sur iOS, l'installation PWA ne fonctionne **qu'avec Safari**.

1. Ouvrez l'URL dans **Safari**
2. Appuyez sur le bouton **Partager** ⎋ (en bas de l'écran)
3. Faites défiler et appuyez sur **"Sur l'écran d'accueil"**
4. Appuyez sur **Ajouter** en haut à droite ✅

---

## 💡 Ce que vous obtenez après installation

| Fonctionnalité | Disponible |

---

## 📲 Comment partager le lien à vos clients

Le comptable peut envoyer le lien par :

- **SMS** : *"Bonjour, envoyez vos factures ici : https://votre-app.streamlit.app"*
- **Email** : lien dans la signature email
- **WhatsApp** : message groupé clients
- **QR Code** : générez un QR code sur [qr-code-generator.com](https://www.qr-code-generator.com) et imprimez-le

---

## 🔗 URL personnalisée (optionnel)

Sur Streamlit Cloud, vous pouvez définir une URL mémorisable :
`https://facturescan-votre-cabinet.streamlit.app`

Dans les paramètres de l'app Streamlit Cloud → **Custom subdomain**.
